О системе PF.Protection

PF.Protection - это современная программная платформа для расчета параметров настройки и мониторинга дифференциально-фазных защит (ДФЗ) в электроэнергетических системах.

Система предназначена для автоматизации процессов расчета уставок защитных устройств, что значительно повышает надежность и безопасность энергосистем.

Ключевые возможности

Расчет параметров ДФЗ

Автоматизированный расчет уставок дифференциально-фазных защит для различных типов оборудования

Моделирование режимов

Имитация различных аварийных и нормальных режимов работы энергосистемы

Анализ чувствительности

Оценка чувствительности защит в различных точках сети

Визуализация результатов

Графическое представление расчетных данных и характеристик срабатывания

Поддержка ЛЭП с ответвлениями

Расчет защит для сложных конфигураций линий электропередачи

Генерация отчетов

Автоматическое формирование технической документации и протоколов испытаний

Технические характеристики

Параметр Значение
Поддерживаемые типы защит ДФЗ-2, ДФЗ-4, ДФЗ-5 и их модификации
Типы ЛЭП Одноцепные, двухцепные, с ответвлениями
Напряжение сети 35 кВ, 110 кВ, 220 кВ, 500 кВ
Точность расчетов ±0.5%
Интерфейс данных COMTRADE, Rastr, Excel, XML
Платформа Windows/Linux, .NET Core 3.1+, Python 3.8+
